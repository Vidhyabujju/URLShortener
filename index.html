DESCRIPTION DU PROJET

En utilisant le framework web de votre choix vous devrez développer une API REST permettant d’une part de « minifier » une URLtransmise en paramètre et d’autre part de récupérer l’URL complète depuis l’URL raccourcie.

Vous développerez une interface minimaliste, plaisante et ergonomique pour visualiser des statistiques sur les URLs produites.

Enfin vous produirez une page HTML correspondant à l’URL minifié et redirigeant sur l’URL originale.
