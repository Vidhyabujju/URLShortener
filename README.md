# URLShortener
 Minify a URL and redirect to Long URL

# Installers
 Python3
 Flask
 Flask-restful

# Project description
 Execute the application with the following command
	$ python restAPI.py
 In a new command window, execute the following command to see the application
        $ curl http://127.0.0.1:5000
